# Projet Conversion rate challenge

## Description du Projet
Il s'agit d'un challenge de machine learning inspiré des compétitions Kaggle.
www.datascienceweekly.org est une newsletter gérée par des data scientists.
Ce projet consiste à comprendre le comportement des visiteurs de leur site.

## Objectifs

* EDA
* Optimisation du f1-score de différents modèles
* Prédiction
* Analyse des paramètres du meilleur modèle et suggestions d'amélioration

## Données

* conversion_data_test.csv: Dataset de test
31 620 lignes, 5 colonnes

* conversion_data_train.csv: Dataset d'entraînement
284 580 lignes, 6 colonnes

## Structure du projet

* ```data```: Jeux de données
* ```Modele/```: Entraînement de modèles
